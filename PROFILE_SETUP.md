# GitHub Profile README 设置说明

这个目录中的 `README.md` 用于上传到 GitHub Profile README 仓库。

## 目标仓库

- GitHub 账号：`zhangxiangbei`
- Profile README 仓库名：`zhangxiangbei`
- 仓库地址应为：`https://github.com/zhangxiangbei/zhangxiangbei`

GitHub Profile README 的规则是：仓库名必须和用户名完全一致，并且仓库必须是 Public。

## 创建步骤

1. 登录 GitHub。
2. 点击右上角 `+`，选择 `New repository`。
3. Repository name 填写：`zhangxiangbei`。
4. Visibility 选择：`Public`。
5. 可以不勾选初始化 README，因为本目录已经准备好了 `README.md`。
6. 创建仓库后，将 `github-profile-readme/README.md` 上传到该仓库根目录。
7. 提交后打开 `https://github.com/zhangxiangbei`，检查个人主页是否显示 README 内容。

## 如果没有显示

请依次检查：

- 仓库名是否为 `zhangxiangbei`，不能多字母、少字母或大小写不一致。
- 仓库是否为 Public。
- `README.md` 是否位于仓库根目录。
- 默认分支是否包含 `README.md`。
- GitHub 页面是否需要刷新或等待几分钟缓存更新。
